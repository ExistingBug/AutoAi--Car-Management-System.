#  AutoAI – Car Management System

A modern full-stack web application that helps car owners manage every aspect of their vehicles from one place.

AutoAI provides secure document storage, maintenance tracking, smart reminders, and an AI-powered car search system, making vehicle management simple, organized, and accessible.

---      

##  Features

###  User Authentication
- Secure user registration & login
- JWT Authentication
- Password encryption using bcrypt
- Refresh Token support
- Protected routes

###  Vehicle Management
- Add multiple vehicles
- Store vehicle details
- Upload vehicle images
- View and manage all registered vehicles

###  Document Management
Store important vehicle documents such as:
- Registration Certificate (RC)
- Insurance
- Pollution Under Control (PUC)
- Driving License
- Other documents

Features include:
- Cloud image storage
- Upload/Delete documents
- Issue & expiry dates
- Notes for each document

###  Maintenance Records
Keep track of your vehicle's servicing:
- Service type
- Service center
- Service date
- Mileage
- Cost
- Description
- Next service date
- Invoice upload

###  Smart Reminders
Never miss important dates:
- Insurance Renewal
- PUC Renewal
- Oil Change
- Vehicle Service
- Custom reminders

Supports:
- Priority Levels
- Repeat schedules
- Completion tracking

###  AI Car Search
Search and compare cars using AI with multiple filters including:
- Brand
- Budget
- Fuel Type
- Transmission
- Body Type
- Seating Capacity
- Mileage
- Engine
- Safety Features
- And many more...
